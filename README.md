## 📊 Student Performance Analytics Dashboard

The **Student Performance Analytics Dashboard** is a React-based data visualization project that displays student marks using different types of charts.

## 🛠 Technologies Used

- **React.js** – Frontend framework
- **Recharts** – Charting library
- **Axios** – API requests
- **JavaScript (ES6+)** – Logic and data processing
- **Tailwind CSS** – Styling

## 🚀 Features

- Visualize student marks using multiple chart types:
  - 📈 Line Chart (Subject comparison)
  - 📊 Bar Chart (Average & subject marks)
  - 📉 Area Chart (Trend visualization)
  - 🥧 Pie Chart (Half-circle / Straight angle)
- Dynamic data loading using:
  - Fetch API
  - Axios
- React Suspense for loading states
- Responsive chart design
- Data transformation (average calculation)
- Clean and modern UI
